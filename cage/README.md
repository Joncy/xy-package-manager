# Beast Package Manager
A npm for IoT<br/>
Command line tool to pop different nodejs directory trees compatible with xway (learn more at http://netbeast.co).<br/>

In the future you will be able to publish and keep version control of your apps published in iot.netbeast.co/market :]

Create an app:
```bash
bpm new myapp
```

Remove:
```bash
bpm rm myapp
```

Piece of cake!<br/>
